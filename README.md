# CRM Uygulaması

React + ASP.NET Core Web API ile geliştirilmiş modern bir Müşteri İlişkileri Yönetimi (CRM) uygulaması.

## 🚀 Kullanılan Teknolojiler

### Backend
- **ASP.NET Core Web API** (.NET 8)
- **Entity Framework Core** (Code First)
- **UnitOfWork** Pattern
- **AutoMapper**
- **DTO'lar** (Data Transfer Objects)
- **SQL Server**

### Frontend
- **React.js**
- **Axios** (API iletişimi)
- **React Router**
- **Bootstrap 

## 📦 Öne Çıkan Özellikler

✅ **Toplu Excel Aktarımı**
- Personel, Müşteri ve Ürün verilerini Excel'den toplu olarak içe aktarabilme
- Excel şablonları ile hızlı veri girişi

✅ **Bildirim Sistemi**
- Kullanıcıya anlık başarı/hata bildirimleri (Toast/Alert)

✅ **CRUD İşlemleri**
- Personel, Müşteri, Ürün yönetimi (Ekleme, Silme, Güncelleme, Listeleme)

✅ **Mimari Desenler**
- UnitOfWork: İşlemleri tek bir çatı altında toplar
- AutoMapper: DTO - Entity dönüşümlerini otomatikleştirir
- DTO'lar: API katmanında veri gizliliği ve güvenliği
